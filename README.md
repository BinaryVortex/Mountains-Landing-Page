# Mountains — Landing Page

Mountains Landing Page Built Using HTML, CSS and JavaScript.

![Landing Page Screenshot](./Screenshot%202024-11-21%20211455.png)

## About

A simple, responsive landing page themed around mountains. This project demonstrates layout, styling, and small interactive behaviors using plain HTML, CSS, and JavaScript — suitable for learning and quick portfolio showcase.

## Preview

See the screenshot above for a preview of the page.

## Features

- Clean, responsive layout
- Hero section with call-to-action
- Smooth styling with CSS
- Small JavaScript interactions (see script.js)

## Built with

- HTML
- CSS
- JavaScript

## Getting started

Clone the repository and open the page locally:

```bash
git clone https://github.com/BinaryVortex/Mountains-Landing-Page.git
cd Mountains-Landing-Page
# Open index.html in your browser (double-click or use a local server)
```

If you prefer a local server (recommended for testing JS/CSS that requires HTTP):

```bash
# Using Python 3
python -m http.server 8000
# Then open http://localhost:8000 in your browser
```

## File structure

- `index.html` — main markup
- `style.css` — styles
- `script.js` — JavaScript interactions
- `Screenshot 2024-11-21 211455.png` — project screenshot used in this README

## Customize

- Edit `index.html` to change content and structure.
- Update `style.css` for appearance and responsive tweaks.
- Modify `script.js` to change behavior or add new interactions.
